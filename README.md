# Aiya Kareem Landing Page

A simple Next.js app for an artist landing page with a modern 70s-inspired palette.

## Setup

1. Place the artist portrait image at `public/aiya-portrait.jpg`
2. Place a parallax background image at `public/parallax-bg.jpg`
3. Run `npm install`
4. Run `npm run dev`

## Notes

- The page uses the warm earth tone palette from the provided color guide.
- The hero section is designed around the portrait image and bold signature heading.
- The vinyl station drop area is a styled prototype for the audio experience.
